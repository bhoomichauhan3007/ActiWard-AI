# 🚀 ActiWard AI

> **From Street Reports to City Action**

ActiWard AI is an AI-powered civic issue reporting platform that helps citizens report local infrastructure issues such as potholes, garbage dumps, broken streetlights, water leaks, and drainage problems. The platform uses Google Gemini AI to analyze uploaded images, classify issues, estimate severity, recommend the responsible department, and assist authorities in resolving problems efficiently.

## 🚀 Installation

```bash
npm install
npm run dev
```

Create a `.env.local` file and add your Gemini API key:

```env
VITE_GEMINI_API_KEY=YOUR_API_KEY
```
